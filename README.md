# DennyPOS Website

Premium Point of Sale System for South African Businesses

## Features

- Modern, futuristic design with glass morphism effects
- 3D animations and parallax scrolling
- Multi-language support (English, Afrikaans, Zulu, Xhosa, etc.)
- Accessibility options (font size, contrast, color blind modes)
- Responsive design for all devices
- WhatsApp integration for sales
- Contact form with email notifications

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion for animations
- React Router for navigation
- React i18next for translations
- PHP backend for contact form

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
