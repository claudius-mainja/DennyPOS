**Version:** 1.0.0

## Common Components

### AccessibilityPanel.jsx
Accessibility features panel providing:
- Font size adjustment (Small, Medium, Large, Extra Large)
- Contrast modes (Standard, High, Low)
- Color blind modes (Protanopia, Deuteranopia, Tritanopia)
- Dark/Light mode toggle
- Language selector

### PageTransition.jsx
Framer Motion page transition wrapper for smooth navigation animations.

### WhatsAppFloat.jsx
Fixed floating WhatsApp button linking to sales chat (+27 74 355 1336).

## Layout Components

### Navbar.jsx
- Fixed header with backdrop blur
- Responsive mobile menu
- Navigation links to all pages

- Logo with hover effects

### Footer.jsx
- Company branding and logo
- Quick links, company links, legal links
